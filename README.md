# discordEmojiStickerCopier
Copies all the emojis and sticker of a targeted server

# How to Use?

`-` Copy the code

`-` Go to Developer Tools

`-` Open Console

`-` Paste the code

`-` Copy ServerID and repace it in serverID var

`-` Click Enter (Copies all the links to clipboard)

`-` Paste it anywhere.
